# Procedures — How I Do Things

## 🎨 Communication Preferences
- User prefers casual, direct, technical communication
- Use emojis for scannable progress updates
- Keep progress updates brief

## 🔧 Tool Workflows
- NAS folder browsing: use nas_list_folder with share-relative paths
- Web research: search_web first, then browse_url on promising results
- Container management: check status before restart/stop

## 📝 Format Preferences
- Use Discord embeds for structured output
- Send as .md file for responses >8000 chars
- Tables for comparisons, bullet lists for summaries

## ⚡ Shortcuts & Patterns
- Docker operations run directly on Mac Mini (no SSH needed)
- NAS operations need SSH on port 24 or FileStation API
- Audiobooks at /Misc/audiobooks on NAS
