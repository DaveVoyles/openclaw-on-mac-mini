🔄 Refresh Patreon Cookies

Follow these steps to refresh your Patreon session cookies:

  1. Open Chrome and navigate to patreon.com
  2. Log in to your Patreon account
  3. Install the EditThisCookie extension (if not already installed)
  4. Click the EditThisCookie icon → Export
  5. Copy the exported JSON
  6. Save to ~/Downloads/cookies.txt
  7. Run: docker cp ~/Downloads/cookies.txt monstervision:/app/cookies/cookies.txt
  8. Downloads will resume automatically within 30 minutes
💡 Tip: Cookies typically last 3-7 days. Refresh when you see yellow/red status.

💭 Recent Features (v0.11.0)

📝 Correction Learning
Correct the bot and it learns a rule. Rules compound over time — the bot never repeats the same mistake. /rules to manage.
👤 User Profile
Auto-learns your preferences, interests, and working style. Profile context personalizes every response. /profile to view.
⏳ Memory Lifecycle
Memories decay when unused, strengthen when accessed. Session handovers persist decisions and next steps automatically.
🧠 Semantic Memory (ChromaDB)
All memories, conversations, and research are embedded into a local vector store. /recall now finds facts even when phrased differently.
📋 Persistent Threads (SQLite)
Unlimited message history, auto-titling, keyword + semantic search via /threads-search. Stale threads auto-archive after 7 days.
🔍 Research Memory
Past research is indexed and searchable via /research-search. Follow-up research builds on prior findings automatically. Completed runs now include persistence receipts (session, vault, vector, and Google Docs when available), and the rerun action schedules a full research cycle every 24h.
🔖 /bookmark Command
Save any URL to the Obsidian vault. Gemini fetches, summarizes, and tags the page automatically.
🤖 Natural-language /ask
Ask for a recap, health report, sports watch guide, box-office report, calendar lookup, or inbox search in plain English. OpenClaw now shortlists the most relevant skills automatically. In allowed channels, plain messages route through default /ask and continue in a bot-owned thread for continuity.
📚 Obsidian Vault
Research reports and bookmarks are saved to data/vault/ as Markdown with YAML frontmatter. Compatible with Obsidian.
📝 /note Commands
Create, browse, view, and search notes directly in the Obsidian vault from Discord via /note create|list|view|search.
🗓 /recap weekly
Summarize the current Discord channel or thread into highlights, action items, or a table. For copy/paste workflows, use /recap copy-latest or /recap copy-thread in the same channel/thread.
🥐 /sports upcoming
Build a live sports watch guide with a markdown table for matchups, times, and where to watch. Ideal for weekly lacrosse or weekend watch lists. Markdown tables are reformatted for Discord; larger/complex tables may include a rendered image attachment.
💬 Channel Role Routing
Assign research, analytics, or bookmarks roles to Discord channels via env vars for specialized AI personas.
⏰ 4 AM Maintenance Cron
Nightly auto-maintenance: git pull skills, restart LLM gateway, backup config to NAS. Vault indexing runs at 3:50 AM.
