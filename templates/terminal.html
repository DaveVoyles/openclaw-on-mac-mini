⚡ Common Workflows

Something seems slow on the Mini

1.ssh davevoyles@192.168.1.93
2.ta — attach to tmux session
3.btop — CPU/mem/network at a glance
4.lzd — see if a container is misbehaving

Run a long agent job on the Mini

1.ssh davevoyles@192.168.1.93 && ta
2.Start the job in one pane
3.Ctrl+A | → split, run oc-logs to monitor
4.Ctrl+A d → detach safely — survives SSH disconnect

Deploy OpenClaw update

1.cdopen
2.gp — push latest
3.oc-restart — rebuild + restart
4.oc-logs — watch startup
